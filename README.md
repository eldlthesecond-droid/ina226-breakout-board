INA226 Breakout board with STM32 Driver

Simple STM32 HAL-based driver for INA226 current/power monitor.
Code example.

Features
I2C communication
Voltage, current, power reading
Calibration support
Simple API
Compatibility

This driver is designed for STM32 HAL-based MCUs and is portable across multiple STM32 families, including (but not limited to):

STM32F0 / F1 / F3 / F4 / F7 / H7 series
STM32 Discovery, Nucleo, and custom boards using HAL I2C drivers

Only requirement is a working STM32 HAL I2C peripheral.

Example Code Note

All example code provided in this repository is written for the STM32F407 Discovery board using the STM32 HAL library.
